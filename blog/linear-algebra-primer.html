Linear Algebra Primer

This article is a fast paced introduction to Linear Algebra. You might benefit from reading it as a little preview to your linear algebra course, or as a review of topics that you have already studied.

Linear algebra is the study of vector space and linear maps. We discuss their properties and representations. There are countless applications in various subjects in computer science, engineering, physics, and (naturally) mathematics. Linear algebra has been cleverly applied to other seemingly unrelated fields as well. To see the power of this subject, it may be worth looking into Google's PageRank Algorithm.

This treatment leaves out details for the sake of speed and understanding, as it is meant to be a quick introduction to the topic. The prerequisites are single variable calculus and set theory, although enough mathematical maturity should get you through most of the content. For a review of integration, see my writeup on integrals.

Many things I want to write about require linear algebra, so I wrote this primarily for people to use it as a reference. The content is based on the books Linear Algebra Done Right by S. Axler and Linear and Geometric Algebra by A. Macdonald.

Vectors

You may have been introduced to the idea behind a "vector" in a general math or physics course. Often, vectors are considered quantities that contain both a "magnitude and a direction," or sometimes "lists of numbers." Both of these definitions may be a sufficient introduction to understanding how vectors are represented, but not necessarily what they are.

Linear algebra axiomatizes the idea of a vector by defining what is called a vector space. If you are not familiar with a "axiom-theorem-proof" style of mathematics, it may take some more time to grasp this material.

Vector Spaces

We can start by creating some definitions. This is a common theme in "abstract" mathematics. Think of it as trying to declare general properties of an object and then proving additional properties that build off of each other.

Scalars in F\mathbf{F}F

Subspaces

A subpsace U\mathbf{U}U of a vector space V\mathbf{V}V is a nonempty subset of V\mathbf{V}V that inherits the properties of scalar multiplication and vector addition from V\mathbf{V}V. The following must be satisfied for u1,u2∈U\mathbf{u}_1, \mathbf{u}_2 \in \mathbf{U}u1​,u2​∈U:

  1. (u1+u2)∈U(\mathbf{u}_1 + \mathbf{u}_2) \in \mathbf{U}(u1​+u2​)∈U (Closed under vector addition)
  2. au1∈Ua\mathbf{u}_1 \in \mathbf{U}au1​∈U (Closed under scalar multiplication)

Note that in order for U\mathbf{U}U to be closed under scalar multiplication, the zero vector must also be in U\mathbf{U}U.

Span and Linear Combinations

A linear combination of a set of vectors v1,…,vn\mathbf{v}_1, \dots, \mathbf{v}_nv1​,…,vn​ in a vector space is defined as

∑i=1naivi\sum_{i=1}^n a_i\mathbf{v}_ii=1∑n​ai​vi​

for scalars {ai}\{a_i\}{ai​}. The span of vectors v1,…,vn\mathbf{v}_1, \dots, \mathbf{v}_nv1​,…,vn​, denoted span(v1,…,vn)\text{span}(\mathbf{v}_1, \dots, \mathbf{v}_n)span(v1​,…,vn​) is the set of all linear combinations of the vectors. We say that a set of vectors SSS spans span(S)\text{span}(S)span(S).

More technically, "set of all linear combinations" means

span(v1,…,vn)={∑i=1naivi:ai∈F}\text{span}(\mathbf{v}_1, \dots, \mathbf{v}_n) = \left\{\sum_{i=1}^n a_i\mathbf{v}_i : a_i \in \mathbf{F}\right\}span(v1​,…,vn​)={i=1∑n​ai​vi​:ai​∈F}

for the appropriate field F\mathbf{F}F.

Linear Independence, Bases, and Dimension

Intuitively, we say that a set of vectors is linearly independent if no vector in the set can be represented as a linear combination of the other vectors. Formally, a set of vectors v1,…,vn\mathbf{v}_1, \dots, \mathbf{v}_nv1​,…,vn​ is linearly independent if and only if for scalars {ai}\{a_i\}{ai​}, the only way to satisfy

a1v1+⋯+anvn=0a_1\mathbf{v}_1 + \cdots + a_n\mathbf{v}_n = \mathbf{0}a1​v1​+⋯+an​vn​=0

is to set ai=0a_i = 0ai​=0 for every iii.

To better understand the intuitive explanation given before, try to write v1\mathbf{v_1}v1​ as a linear combination of the other vectors:

a1v1+⋯+anvn=0a_1\mathbf{v}_1 + \cdots + a_n\mathbf{v}_n = \mathbf{0}a1​v1​+⋯+an​vn​=0
a1v2=−a2v2−⋯−anvna_1\mathbf{v}_2 = -a_2\mathbf{v}_2 - \cdots - a_n\mathbf{v}_na1​v2​=−a2​v2​−⋯−an​vn​

Dividing by a1a_1a1​ is impossible because a1a_1a1​ must be equal to 000 for this to be true. The same argument can be made for v2,…,vn\mathbf{v}_2, \dots, \mathbf{v}_nv2​,…,vn​.

Bases

A basis for a subspace U\mathbf{U}U is any linearly independent set of vectors that span U\mathbf{U}U. In Exercise 1.8, the set of vectors SSS is a basis for U\mathbf{U}U. The same exercise also proves that any basis has exactly nnn vectors.

This is a phenomenal development. The idea of a basis is essential and will come up many times both in and outside of linear algebra because it allows us to refer to a specific vector in a subspace without reference to a coordinate system, i.e. we can refer to a vector as a mathematical object rather than in terms of its coordinates. We will discuss the former in the next section, but the definition of "dimension" is provided below.

Dimension

Let a basis for a vector space V\mathbf{V}V have nnn vectors. We say that dim⁡(V)=n\dim(\mathbf{V}) = ndim(V)=n.

Exercise 1.9: Let C[a,b]C[a, b]C[a,b] represent the set of all functions f:[a,b]⊂R→Rf : [a, b] \subset \mathbf{R} \to \mathbf{R}f:[a,b]⊂R→R. Does C[a,b]C[a, b]C[a,b] have a finite basis?
