Writing

Here are some things that I’ve written for fun or to strengthen my own knowledge. Check them out, hopefully they are useful to you. If the link does not work, it most likely means that I’m not done writing.

Intro to Stone Duality Proofs

Proofs for the theorems in Prof. Alexander Kurz's notes introducing Stone Duality. Currently being updated section-by-section.

Understanding Neural Networks

A comprehensive article about neural networks and how they work (mathematically), complete with an implementation of neural nets in python.

Understanding L'Hospital's Rule

Presenting the motivation, proof, and applications of L'Hospital's rule and other related topics, like Rolle's theorem and the mean value theorem.

The Irrationality of Pi

A presentation of the proof that $\pi$ is irrational using techniques familiar to calculus students, with a formal introduction to rational/irrational numbers.

Decentralized Apps & Blockchain

An introduction to decentralized applications and blockchain technology, with emphasis on Ethereum and Bitcoin.

Linear Algebra Reference

A reference (and some exercises) for topics in Linear Algebra based on Axler's and MacDonald's Linear Algebra texts.

My Projects

Here are some projects I've worked on. Most of them are open sourced and can be found on my github. If the link does not work, it probably means I am not done working on it.

Candl

A tiny, pedagogical neural network library with a pytorch-like API and tensor-valued autodiff.

Accent.js

An open source web development framework made in less than 10kb built for modularity.

Feedbump

A tool for businesses to give and receive feedback from their customers, used by ~100 businesses to collect and manage their customer feedback.

Firenas

A firebase-powered “network attached storage” (NAS) for storing and accessing files from a remote filesystem using your terminal.

This Blog Template

The template which I made to create this website. Features markdown blogging, build/deploy tools, and global $\LaTeX$ support.

Cammy

A markdown editor made with Electron that uses natural language processing to automatically detect and schedule tasks, events, etc.
