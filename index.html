Projects

Here are some of my favorite projects. Most of them are open sourced and can be found on my github. If the link does not work, it probably means I am not done working on it.

Candl

A tiny, pedagogical neural network library with a pytorch-like API. Complete with a tensor-valued autodiff API, candl allows you to build and train simple neural networks.

Accent.js

An open source web development framework made in less than 10kb. By splitting up pieces of the framework into modules, accent developers can choose exactly what it is they want to use for their project, instead of having excess features that they would never use anyways.

Feedbump

A tool for businesses to give and receive feedback from their customers. Whether you want to make a review platform, private feedback form, or issue tracking board, Feedbump gives you all the tools you need.

Firenas

A firebase-powered “network attached storage” (NAS) for storing and accessing files from a remote filesystem using your terminal. One of the features (and my favorite) is the ability to open files locally without actually keeping them locally.

Cammy

A markdown editor that uses natural langauge processing to automatically detect and schedule taks, events, etc. The editor is built with Electron and can be downloaded natively on Linux, Darwin, and Windows systems.

Interesting Resources

Here are some things that I find interesting. Most of these are just random projects (repositories), books, papers, articles, etc.

Little Man Computer

A visualization of assembly language instructions on a basic Von-Neuman Computer. The instructions are clearly depicted on a view of the computer's essential components. Custom assembly language instructions can also be inputted, making this a great educational tool.

Manim

An animation engine for mathematics. It can be used to make really cool educational math and science videos. 3Blue1Brown uses this software for the mathematics visualizations.

Mafs

A math visualization library for the web (distributed as React components). Mafs is especially useful for interactive on-site graphs and visuals.

d2l.ai

An excellent deep learning textbook used in many universities worldwide. The book treats the concepts of machine learning from the ground up, with little prior knowledge in machine learning required. (Calculus, Probability, Linear Algebra, and Programming are very commonly used, however)
